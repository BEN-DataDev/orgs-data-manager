Information
This Online Register allows you to search for NSW Incorporated Associations, Incorporated Limited Partnerships and Limited Partnerships. Limited data is available for Cancelled entities.

If the entity you are searching for does not appear in search results you can complete an online inquiry at https://www.fairtrading.nsw.gov.au/resource-library/forms/associations-enquiry-form.

If you wish to search for a Co-operative, please click on the link: https://applications.fairtrading.nsw.gov.au/coopsregister/
Organisation Search
Organisation Name allows you to search on the Organisation’s current or past names.
Organisation Type allows you to search for all Organisations of a particular type (Association, ILP or LP).
Organisation Number allows you to search on the Organisation’s Number.
Location allows you to search for all Organisations whose Official / Registered Office Address is within the entered Suburb.
Postcode allows you to search for all Organisations whose Official / Registered Office Address is within the entered Postcode
Organisation Status allows you to search for all Organisations whose current status is the selected status.
Incorporated Limited Partnership Sub Class Type allows you to search for Incorporated Limited Partnerships Sub Class Type.
Date Registered allows you to search for all Organisations where their date registered falls within the selected period.
Date Cancelled allows you to search for all Organisations where their date cancelled falls within the selected period.
